# This is the second revision of the 'wew' language, still pretty much C but worse

# What works:
+ Parsing
+ Generating ast and objects

# What isn't implemented
+ Compilation
+ Code generation


# example:

```
int *main(int that, int *** this) {
  if (1) {
    print(this);
    while (i < 2) {a := 2;}
  } else {
	*b := dothis(1*2, that(4+test, else));
  }
}

```
